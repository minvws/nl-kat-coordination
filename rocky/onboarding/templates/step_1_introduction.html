{% translate "How does OpenKAT work" %}

{% blocktranslate trimmed %} OpenKAT is able to give insight into security risks of your organization. Such as your websites, mail servers, databases or online data (leaks), outdated software, etc. {% endblocktranslate %}

{% blocktranslate trimmed %} OpenKAT uses plugins to find and assess the area's that might be at risk and reports these back to you. Each plugin has its own skillset which could be scanning, normalizing or analyzing data. As a user you decide which areas you would like to monitor or scan and which insight you would like to receive. {% endblocktranslate %}

{% blocktranslate trimmed %} Within OpenKAT you can view the insights as well as all the data OpenKAT has found. You can choose to browse through the data or view reports. {% endblocktranslate %}

{% blocktranslate trimmed %} During this introduction you will be guided through the steps to create a DNS report. {% endblocktranslate %}
